# Ethiopia Surveillance IG (sandbox demo subset)

A minimal surveillance IG derived from [`et.fhir.core`](https://moh-ethiopia.github.io/ETBase/) for
the [hapi-sandbox](https://github.com/pmanko/hapi-sandbox) conformance sandbox. Carries the
surveillance extensions (and disabled STI/VMMC/SRH content); enables what the HIV IG and the
medication-dispense demo need. Builds via the IG Publisher; CI publishes to GitHub Pages.
